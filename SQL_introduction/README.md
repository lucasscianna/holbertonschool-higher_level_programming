# SQL - Introduction 🗄️

Ce projet fait partie du cursus **Holberton School**. Il pose les bases de la manipulation de données avec **MySQL**. L'objectif est de comprendre comment créer des bases de données, gérer des tables et effectuer des opérations CRUD (Create, Read, Update, Delete) via des scripts SQL.

## 📝 Description

Ce dépôt contient une série de scripts SQL permettant de :
- Créer et supprimer des bases de données et des tables.
- Insérer, mettre à jour et supprimer des données.
- Effectuer des requêtes d'analyse (tris, regroupements, moyennes).
- Gérer les contraintes de données (`IF NOT EXISTS`, `IS NOT NULL`).
