# JavaScript - Warm up

## Background Context
JavaScript is used for many things. In this project, we explore JavaScript for two main reasons:
1.  **Scripting**: Similar to our approach with Python.
2.  **Web front-end**: Preparing for dynamic integration in the AirBnB project.

For now, we focus on scripting to master the basic concepts of the language before moving towards JQuery and dynamic web development.

## Learning Objectives
At the end of this project, you should be able to explain:
* Why JavaScript programming is amazing.
* How to run a JavaScript script using Node.js.
* The differences between `var`, `const`, and `let`.
* Available data types and variable scope.
* How to use loops (`while`, `for`) and control flow (`if`, `else`, `break`, `continue`).
* How to define and use functions (and what they return by default).
* Arithmetic operators and dictionary manipulation.
* How to import files.

## Requirements
### General
* **Editors**: `vi`, `vim`, `emacs`.
* **Environment**: Ubuntu 20.04 LTS using **Node.js (version 14.x)**.
* **Formatting**:
    * All files must end with a new line.
    * The first line must be exactly: `#!/usr/bin/node`.
    * Code must be **semistandard** compliant (Standard Rules + semicolons).
* **Execution**: All files must be executable (`chmod +x`).
* **Testing**: File length will be tested using `wc`.
