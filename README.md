# 🐍 Higher Level Programming - Python & Algorithms
> **Maîtrise des fondamentaux de Python et des structures de données pour le Génie Logiciel.**

Ce dépôt regroupe l'ensemble de mes travaux sur le langage **Python**. Il constitue le socle technique indispensable pour mes futurs projets en **Intelligence Artificielle**.

## 🛠️ Compétences Développées
- **Fondamentaux :** Types de données, structures de contrôle, listes et dictionnaires.
- **POO (Programmation Orientée Objet) :** Conception de classes, héritage, et encapsulation.
- **Gestion de Données :** Lecture/écriture de fichiers, sérialisation JSON CSV , apprentissage SQL.
- **Algorithmique :** Optimisation de code et résolution de problèmes complexes.

## 📈 Objectif : Intelligence Artificielle
La rigueur acquise dans ces projets me permet d'aborder sereinement l'implémentation de modèles de Machine Learning et le traitement de données massives.
